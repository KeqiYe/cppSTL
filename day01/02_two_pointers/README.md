# 第二节：双指针

双指针不是新的 STL 容器，而是同时使用两个下标处理数组或字符串。

```cpp
int left = 0;
int right = static_cast<int>(nums.size()) - 1;

while (left < right) {
    // 使用 nums[left] 和 nums[right]
}
```

两个下标通常有两种移动方式：从两端向中间移动，或者一快一慢向同一方向移动。本节两题都使用“左右夹逼”。

写双指针前必须先说清楚：两个指针分别表示什么、为什么可以移动某一侧、移动后不会错过答案。只有两个变量 `left/right`，但无法解释移动依据，不算真正掌握。

本节会使用这些 `vector` 知识：

```cpp
nums.size();                    // 返回无符号的 size_t
nums[index];                   // O(1) 随机访问，不检查越界
std::sort(nums.begin(), nums.end());
```

第 11 题学习“根据不会更优的一侧排除答案”。第 15 题学习“排序后固定一个数，再用左右指针寻找另外两个数”，同时练习去重。

完成标准：能够解释第 11 题为什么移动短边；能够独立写出第 15 题对 `i`、`left`、`right` 三处的去重逻辑；能够分析为什么双指针把两层枚举降为一层。
